# Weed and Read

Privacy-first cannabis boutique storefront for a Victorian reading-room speakeasy in Albuquerque, New Mexico.

## Stack

- Next.js (App Router) + React
- Tailwind CSS v4
- TypeScript

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project rules

Cursor rules in `.cursor/rules/` cover privacy (no tracking, age gate via `localStorage`, minimal newsletter schema), the brand design system, and New Mexico cannabis advertising compliance (NMAC 16.8.3.8).
